# 📊 Python-MatrizConfusao

A Python project for evaluating classification models using confusion matrices, built with Jupyter Notebook. Demonstrates how to visualize and interpret model performance metrics.

## 📱 About The Project

Python-MatrizConfusao is a simple and practical implementation of confusion matrix analysis. The project uses the Titanic dataset to evaluate a classification model's predictions, showing true positives, false positives, true negatives, and false negatives.

This project was created for study purposes and to demonstrate model evaluation techniques using Python.

## ✨ Features

- Confusion matrix generation and visualization
- Classification model performance evaluation
- Titanic dataset example
